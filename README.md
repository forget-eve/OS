### 关于作业

- 作业有课本题，还有其他考研题，每周不一定有作业，看周进程，一般15周有10~12周作业，网上基本都有答案。

- 大作业（实验）不超过6次，大概框架网上有参考，但是实验很重要，大部分代码用于理解，核心代码自己填写，同时要有大部分debug。

- 对于迟交作业，不允许迟交，交作业时间当周周日，在bb系统上发布作业and提交作业（pdf手写，中英文都行）。
